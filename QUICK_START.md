# Quick Start Guide

## Installation

1. **Install Node.js** (if not installed)
2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Start Server**:
   ```bash
   npm start
   ```

## Access

- **Website**: http://localhost:3001
- **Admin Panel**: http://localhost:3001/admin
- **Password**: `admin123`

## Basic Usage

### Adding Products
1. Go to admin panel
2. Login with password
3. Click "Add Product"
4. Fill details and upload image
5. Save product

### Managing Orders
1. Orders appear automatically when customers buy
2. Export to Excel for accounting
3. Track stock levels

### Configuration
Edit `.env` file for:
- Email settings
- Admin password
- Other configurations
